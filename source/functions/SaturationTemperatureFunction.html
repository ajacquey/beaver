SaturationTemperatureFunction

This function computes saturation temperature from a given pressure function : where the function is provided via the T_sat(p) API of objects deriving from TwoPhaseFluidProperties.

Input Parameters

  • fp_2phase2-phase fluid properties

    C++ Type:UserObjectName

    Controllable:No

    Description:2-phase fluid properties

  • pPressure function

    C++ Type:FunctionName

    Controllable:No

    Description:Pressure function

Required Parameters

  • execute_onLINEARThe list of flag(s) indicating when this object should be executed, the available options include NONE, INITIAL, LINEAR, NONLINEAR, TIMESTEP_END, TIMESTEP_BEGIN, MULTIAPP_FIXED_POINT_END, MULTIAPP_FIXED_POINT_BEGIN, FINAL, CUSTOM.

    Default:LINEAR

    C++ Type:ExecFlagEnum

    Options:NONE, INITIAL, LINEAR, NONLINEAR, TIMESTEP_END, TIMESTEP_BEGIN, MULTIAPP_FIXED_POINT_END, MULTIAPP_FIXED_POINT_BEGIN, FINAL, CUSTOM

    Controllable:No

    Description:The list of flag(s) indicating when this object should be executed, the available options include NONE, INITIAL, LINEAR, NONLINEAR, TIMESTEP_END, TIMESTEP_BEGIN, MULTIAPP_FIXED_POINT_END, MULTIAPP_FIXED_POINT_BEGIN, FINAL, CUSTOM.

    Description:Adds user-defined labels for accessing object parameters via control logic.
