SpecificEnthalpyAux

Computes specific enthalpy from pressure and temperature

The specific enthalpy is computed directly from the fluid properties object specified. This, with additional postprocessing, can be used to evaluate the amount of thermal energy stored in a fluid system.

    C++ Type:std::vector<VariableName>

    Controllable:No

    Description:Temperature

  • fpThe name of the user object for fluid properties

    Description:The name of the user object for fluid properties

  • pPressure

    C++ Type:std::vector<VariableName>

    Controllable:No
